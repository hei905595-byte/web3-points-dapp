# UI Reference Images

Place old UI screenshots and design references in this folder.

Recommended naming:

- `01-home-desktop.png`
- `02-wallet-modal.png`
- `03-task-actions.png`
- `04-mobile-home.png`

Supported formats:

- PNG
- JPG / JPEG
- WebP
- GIF

Include the button name and purpose in the filename or a nearby text file when
possible.
